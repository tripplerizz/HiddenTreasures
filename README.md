# Hidden Treasures

Kevin Nguyen
Hector Rizo

An app to find cool local places wherever you travel!

- To run:
  - Inside the root directory of the project, create a `local.properties` file
  - Add your Google Maps API key to the file: `MAPS_API_KEY="YOUR_API_KEY_HERE"`
